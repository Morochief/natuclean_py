# DESIGN.md

## Visual Style
- **Theme:** Zen Minimalist + Luxury
- **Primary Color:** #19e63c (Vibrant Green for sustainability)
- **Background:** #fdfcf9 (Off-white, warm)
- **Text:** #2c2c2c (Deep charcoal)
- **Accent:** #d4a373 (Craft brown/gold)
- **Typography:**
  - Headers: 'Playfair Display', serif (bold, elegant)
  - Body: 'Inter', sans-serif (clean, modern)

## Components
- **Buttons:** Rounded (1rem radius), subtle hover lift, primary is solid green/charcoal.
- **Cards:** Glassmorphism effect (blur), soft shadows, micro-animations on hover.
- **Layout:** Generous whitespace (spacing-xl), centralized container (1200px).
- **Effects:** Smooth fade-ins (`animate-fade-in`), scale on hover.

## "Elite" Features
- Use `shadcn-ui` components where possible for accessibility and polish.
- Solid.js for interactive islands.
- Images should be high-resolution, centered, with soft lighting.
